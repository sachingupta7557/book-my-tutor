# Book-my-tutor using MERN Stack

BookMyTutor is a full-stack web application that connects students with qualified tutors for different subjects.  
Users can browse tutors, view profiles, and book sessions easily through an interactive and user-friendly interface.

## Features

* User Authentication (Login/Signup)
* Tutor Listing with Subject
* Booking system for scheduling classes
* Fully responsive UI
* Admin panel to manage tutors and users
* admin panel allows the admin to add tutors to the platform
* Students can make secure online payments for their sessions
* Secure payment integration using Razorpay

## Installation and Setup

 1. Clone the repository to your local machine
     ```bash
     git clone https://github.com/sachingupta7557/book-my-tutor.git

 2. Install dependencies for frontend,backend and admin
    ``` bash
   
   cd BOOKMYTUTOR

   Start backend
             cd backend
             npm install

   Start frontend
             cd frontend
             npm install

    Start admin
             cd admin
             npm install
 
 3. Run the Project
    ```bash
    
    Start backend
       cd backend
       npm start

    Start frontend
       cd frontend
       npm start
   
    Start admin
       cd admin
       npm start

4. Access the application in your web browser for frontend at http://localhost:5173

5. Access the application in your web browser for admin at  http://localhost:5174
